Proyectos

App. Tareas pendientes

Aplicación web simple que permite a los usuarios gestionar sus tareas diarias.


Tecnologías usadas:


HTML5, CSS3, JavaScript, Node.js, Express, MongoDB.


Desafíos:


Implementar la persistencia de datos para que las tareas se guarden y carguen correctamente entre sesiones.


Soluciones:


- Se utilizó MongoDB para almacenar las tareas de manera persistente, y Express para manejar las rutas del servidor y las operaciones CRUD.


- La interfaz fue diseñada con HTML y CSS, utilizando JavaScript para actualizar dinámicamente la lista de tareas sin recargar la página.

App. Calculadora de presupuestos

Aplicación que permite a los usuarios gestionar sus ingresos y gastos mensuales, generando un balance y gráficos de su situación financiera.


Tecnologías usadas:


React.js, CSS3, Node.js, Express, SQLite.


Desafíos:


Crear un sistema de categorización para los ingresos y gastos que sea flexible y fácil de usar.


Soluciones:


- Se utilizó React.js para desarrollar una interfaz modular y reactiva, permitiendo a los usuarios interactuar con los datos en tiempo real.


- Para los gráficos, se integró la librería Chart.js, lo que facilitó la creación de representaciones visuales atractivas de los datos financieros.

Sitio web de portafolio personal

Sitio web que presenta el portafolio de un desarrollador, incluyendo su biografía, habilidades, proyectos destacados, y un formulario de contacto.


Tecnologías usadas:


HTML5, CSS3, JavaScript.


Desafíos:


Diseñar un sitio web responsivo e implementar un formulario de contacto funcional que valide los datos del usuario.


Soluciones:


- Se utilizó CSS Flexbox y Grid para crear un diseño responsivo que se adapta a diferentes tamaños de pantalla.


- El formulario de contacto fue validado usando JavaScript.
